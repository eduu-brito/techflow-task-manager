# TechFlow Task Manager - Logística Ágil

Sistema básico de gerenciamento de tarefas desenvolvido para otimização de fluxos de trabalho em startups de logística.

## 🚀 Metodologia Adotada
Utilizamos o framework **Kanban** integrado diretamente ao ecossistema do GitHub para monitorar o fluxo de trabalho em tempo real, priorizar demandas críticas e garantir entregas contínuas e previsíveis.

## 🔄 Gestão de Mudanças (Alteração de Escopo)
**Justificativa:** Durante o alinhamento com a startup de logística, identificou-se que apenas listar tarefas não resolvia o problema de gargalo nas entregas.

Foi solicitada uma alteração emergencial no escopo para incluir uma flag de Prioridade Crítica nas tarefas, permitindo destacar despachos urgentes de cargas. O backlog foi adaptado no GitHub Projects e a funcionalidade foi concluída.
